# @takeyaqa/pict-wasm

## Project overview

Unofficial WebAssembly build of [Microsoft PICT](https://github.com/microsoft/pict).  
This package works in both **Node.js** and **browsers**.

WebAssembly is built from the forked C++ code using Emscripten. The code for calling WebAssembly and its tests is developed separately and managed in the following directories:

- `wasm-src`
- `wasm-test`

## Setup commands

- Setup emsdk: `./install_emsdk.sh`
- Install deps: `pnpm install`
- Build WebAssembly: `pnpm run build`
- Format code: `pnpm run fmt`
- Run lint: `pnpm run lint`
- Check type: `pnpm run typecheck`
- Run tests: `pnpm run test:run`

## Development instructions

- Always run format, lint, type checking, and tests before committing.

## Review instructions
